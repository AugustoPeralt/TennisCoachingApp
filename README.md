# 🎾 TenisCoaching - Plataforma de Gestión de Entrenamientos

TenisCoaching es una solución integral diseñada para optimizar la interacción entre entrenadores de tenis y sus alumnos. La plataforma permite la gestión de clases, seguimiento del progreso técnico y coordinación de calendarios en tiempo real.

## 🚀 Sobre el Proyecto
👥 Mi Contribución Técnica y Liderazgo
En este proyecto trabajé directamente con el fundador y desarrollador principal, actuando como pilar técnico en momentos críticos y liderando la evolución de la arquitectura.

Arquitectura de Datos y Migración: Lideré la migración estratégica de Firebase a Supabase, optimizando la estructura de la base de datos y mejorando la gestión de la lógica del backend.

Sistema de Recomendaciones Inteligentes: Ideé e implementé la lógica de planificación de entrenamientos. El sistema analiza la carga semanal del jugador para sugerir ejercicios específicos, personalizando la experiencia del atleta.

Desarrollo Mobile (iOS): Responsable del proceso de empaquetado y adaptación para dispositivos móviles, gestionando actualmente el despliegue en la Apple App Store.

Funcionalidades de Dashboard: Desarrollé el panel de control administrativo, permitiendo una gestión fluida de las métricas del negocio y el seguimiento de los alumnos.

Resolución de Problemas (Troubleshooting): Actué como soporte técnico avanzado para el desarrollador principal, resolviendo bloqueos de lógica y bugs complejos en el sistema.
